## ğŸ’» Deskripsi Proyek

Proyek ini merupakan **Landing Page Responsif** untuk sebuah perusahaan teknologi fiktif bernama **"TechCo."**. Landing Page ini dibangun dengan tujuan mendemonstrasikan penguasaan terhadap konsep *responsive design* dan penggunaan komponen dari CSS Framework.

* **Tampilan Responsif:** Tampilan optimal di berbagai ukuran perangkat (Desktop, Tablet, dan Mobile).
* **Struktur Komponen:** Menggunakan komponen standar untuk navigasi, *hero section*, kartu fitur, dan formulir kontak.
* **Formulir Kontak:** Implementasi penuh dari *styling* komponen Formulir yang dipelajari pada Percobaan 3.

## ğŸ“ Komponen dan Implementasi

Proyek ini mencakup semua bagian wajib dari penugasan:

1.  **Header/Navigation:** Menggunakan komponen **Navbar** Bootstrap dengan *toggler* untuk mode mobile.

6.  **Footer:** Berisi informasi Hak Cipta (*Copyright*) dan tautan media sosial.
